# QuoteQuick - UK Tradesman Quote Builder

A Chrome extension for UK tradesmen to quickly generate professional quotes on MyBuilder, Checkatrade, and other trade platforms.

## Features

- **Template Library**: Save and reuse quote templates
- **Variable Substitution**: Auto-fill customer name, job type, and location
- **One-Click Copy**: Copy formatted quotes to clipboard instantly
- **Trade-Specific**: Tailored for UK trades (plumbers, electricians, gas engineers, etc.)
- **Works On**: MyBuilder, Checkatrade, RatedPeople, TrustATrader, MyJobQuote

## Installation (Development)

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select the `quote-quick` folder
5. The extension icon should appear in your toolbar

## Usage

1. Click the QuoteQuick icon in your toolbar
2. Browse your saved templates or create a new one
3. Visit a job lead on MyBuilder/Checkatrade
4. Click "Copy Quote" - variables are auto-filled from the page
5. Paste into your reply!

## Creating Icons

The extension needs icons in the `icons/` folder:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

You can create these using any image editor or online tool. A simple hammer/wrench icon works well.
